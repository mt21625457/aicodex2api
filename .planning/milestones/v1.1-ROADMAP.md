# Roadmap: Bulk Edit Productivity (v1.1)

## Milestones

- ✅ **v1.0 OpenAI Bulk Edit Parity** — shipped 2026-02-28 (archive: `.planning/milestones/v1.0-ROADMAP.md`)

## Overview

本里程碑聚焦“批量编辑效率提升”。在现有 scoped bulk-edit 基础上，先补齐可视化范围预览以降低误选风险，再提供模板复用能力，最后补齐质量门禁与上线验收。

**Phase Numbering:**

- Decimal phases (4.1, 4.2) reserved for urgent insertions

- [x] **Phase 5: Template Save & Reuse** - 支持保存批量编辑模板并复用
- [x] **Phase 6: Quality Gate & Rollout (v1.1)** - 测试、覆盖率、UAT 与交付说明

## Phase Details

### Phase 4: Scope Preview & Guardrails UX
**Goal:** 在进入批量编辑弹窗前提供清晰的 `platform:type` 分组统计和限制提示，减少误操作。
**Depends on:** Existing scoped bulk-edit flow
**Requirements**: [BULK-09]
**Success Criteria** (what must be TRUE):
  1. 管理员可看到每个 `platform:type` 分组数量与当前选择范围。
  2. 不可编辑范围有明确禁用态与提示文案。
  3. 选择范围信息在 scoped 弹窗中可追溯。
**Plans**: 1 plan

Plans:
- [x] 04-01: 范围统计数据模型 + 入口 UI + i18n

### Phase 5: Template Save & Reuse
**Goal:** 允许管理员将批量编辑配置保存为模板，并在后续操作中快速套用。
**Depends on:** Phase 4
**Requirements**: [BULK-10]
**Success Criteria** (what must be TRUE):
  1. 管理员可保存当前批量配置为命名模板。
  2. 模板可在同范围类型下快速应用并预览变更。
  3. 模板应用仍遵循“勾选才发送”语义，避免误覆盖。
**Plans**: 1 plan

Plans:
- [x] 05-01: 模板 schema + 前端模板面板 + localStorage 模板持久化

### Phase 6: Quality Gate & Rollout (v1.1)
**Goal:** 为 v1.1 新增能力补齐测试与验收闭环，确保可发布。
**Depends on:** Phase 5
**Requirements**: [BULK-11]
**Success Criteria** (what must be TRUE):
  1. 前后端新增逻辑有单元测试覆盖，核心改动覆盖率 ≥85%。
  2. 模板保存/应用与范围预览关键路径有自动化验证。
  3. 输出可复用 UAT 清单（成功/失败/回滚/权限边界）。
**Plans**: 1 plan

Plans:
- [x] 06-01: 测试补齐 + 覆盖率报告 + UAT 文档
